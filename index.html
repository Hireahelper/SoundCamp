SoundCamp

SoundCamp is a chrome extension that adds more sounds to Campfire chat sessions.

Install »

What it is

This is a a Chrome browser extension. It provides additional sounds for Campfire chat sessions. A button is added to the Campfire room interface, and upon clicking it a menu is presented. Clicking on an option in this menu will result in a sound being heard by members of the chat room who have this plugin installed.

Why

The sounds in Campfire are fun, but limited. We wanted to have more. Sounds can also be useful. We have a number of automated systems in place that we feed into Campfire via our bot. Now our bot can provide different audible alerts.
